# QQ经典农场自动巡查脚本说明

## 1. 项目简介

该脚本用于在 macOS 环境下自动巡查 `QQ经典农场` 窗口，通过 OCR 识别界面文字并执行自动点击，完成常见的收获、浇水、除草、除虫，以及异常界面（如重新登录/重新连接）的恢复操作。

脚本入口文件为：`main.py`。

## 2. 使用到的技术

- Python 3
- `Quartz`（macOS CoreGraphics）：窗口枚举与窗口截图
- `Vision`（macOS Vision Framework）：文字识别（OCR）
- `AppKit`：激活并聚焦目标窗口
- `OpenCV (cv2)`：图像格式转换与尺寸缩放
- `NumPy`：图像缓冲区处理
- `pyautogui`：模拟鼠标点击
- `logging`：运行日志输出

## 3. 核心功能

- **窗口捕获**：按窗口标题匹配 `QQ经典农场`，抓取当前窗口画面。
- **文字识别**：通过 Vision 对截图进行 OCR，识别中文和英文。
- **任务分类与优先级**：
  - 故障恢复关键词：`重新登录`、`重新连接`（最高优先级）
  - 农场操作关键词：`可摘`、`一键收获`、`一键浇水`、`一键除草`、`一键除虫`
- **坐标映射与点击策略**：
  - 恢复任务：点击识别文字框中心
  - 农场任务：点击文字框上方偏移位置（避免直接点到文字区域）
- **循环巡查**：按固定间隔持续运行，支持 `Ctrl+C` 安全退出。

## 4. 脚本执行流程

1. 按窗口标题查找目标窗口并截图。
2. 对截图进行 OCR，拿到文本与对应边界框。
3. 根据关键词进行匹配（含一定容错逻辑）。
4. 将识别到的边界框坐标映射为屏幕坐标。
5. 若有任务，先激活目标窗口。
6. 先执行恢复任务，再执行农场任务。
7. 等待下一轮巡查。

## 5. 关键配置项（`main.py`）

- `FIXED_DELAY_MS`：每次点击后的固定延时（秒）
- `RECOVERY_WAIT`：故障恢复点击后额外等待时间（秒）
- `LOOP_INTERVAL`：每轮巡查之间的间隔（秒）
- `TARGET_SIZE`：截图后统一缩放尺寸（影响 OCR 结果）
- `RECOVERY_KEYWORDS`：恢复任务关键词列表
- `FARM_ACTION_KEYWORDS`：农场操作关键词列表

> 建议优先调整 `TARGET_SIZE` 与各类延时参数，以适配不同分辨率和机器性能。

## 6. 运行环境要求

- 操作系统：macOS（依赖 Quartz / Vision / AppKit）
- Python 版本：建议 Python 3.9+
- 需要授予相关系统权限：
  - 屏幕录制权限（用于窗口截图）
  - 辅助功能权限（用于模拟点击）

## 7. 安装依赖

先创建并激活虚拟环境（可选），再安装依赖：

```bash
pip install opencv-python numpy pyautogui pyobjc
```

## 8. 运行方式

在项目目录执行：

```bash
python main.py
```

运行后会持续循环巡查，终端会打印中文日志。按 `Ctrl+C` 停止。

## 9. 日志与排错建议

- 如果提示“未找到标题包含 `QQ经典农场` 的窗口”：
  - 检查窗口标题是否变化
  - 确认目标窗口在前台且可见
- 如果 OCR 识别不稳定：
  - 调整 `TARGET_SIZE`
  - 保证界面清晰、无遮挡
- 如果无法自动点击：
  - 检查 macOS 辅助功能权限是否已授予
  - 尝试降低系统缩放或调整窗口位置

## 10. 注意事项

- 本脚本基于界面文字识别和屏幕坐标点击，对分辨率、缩放比例、窗口布局较敏感。
- 关键词策略为规则驱动，界面文案变化时需要同步更新关键词。
- 请仅在你可控的环境中使用自动化脚本。
