# QQ经典农场自动巡查脚本系统

## 1. 项目简介

本系统是专为 macOS 环境设计的 `QQ经典农场` 自动化工具。它利用先进的 OCR 文字识别和图像处理技术，实现全自动的农场管理，包括主场维护、好友互助、仓库交易以及智能播种。

项目采用了模块化设计，结构清晰，易于扩展和维护。

## 2. 核心模块说明

- **`main.py` (主程序入口)**:
  - 核心控制流：负责主场巡查循环、任务调度与分发。
  - 异常处理：监控并处理“重新登录”、“重新连接”等系统级对话框。
  - 状态管理：统计巡查次数，定期触发好友场巡查与清理任务。
  
- **`friend.py` (好友农场模块)**:
  - 实现拜访好友、巡查好友家农场、执行互助操作（浇水、除草、除虫、摘取）。
  - 智能切换：支持在多位好友间自动轮换，并能准确识别“回家”路径。

- **`trading.py` (交易与物流模块)**:
  - **自动出售**: 自动进入仓库执行“批量出售”果实，确保持续的经济收益。
  - **自动补货**: 当种子库存耗尽时，自动前往商店检索并购买指定的种子（如“迎春花”）。

- **`oc.py` (视觉识别核心)**:
  - **土地检测**: 基于 OpenCV 模板匹配，精确识别 24 块土地的坐标及其状态（空闲、已种植）。
  - **多级土地识别**: 支持黑土地、红土地、金土地的特征识别。
  - **种子识别**: 在播种界面使用多尺度匹配（Multi-scale matching）精准定位指定种子的坐标。

## 3. 技术路线

- **窗口管理**: 使用 `Quartz` 窗口枚举技术定位游戏窗口。
- **图像截取**: 实时捕获游戏画面并转换为 OpenCV 格式进行后续处理。
- **文字识别 (OCR)**: 调用 macOS 原生 `Vision` 框架，支持高精度中英文识别。
- **鼠标模拟**: 使用 `pyautogui` 实现人性化的点击、拖拽及坐标偏移控制。
- **日志系统**: 全中文多级日志输出，记录巡查全程及异常详情。

## 4. 关键功能亮点

- **全图满扫播种**: 识别空土地后，利用鼠标点击与拖拽组合技，实现 1-24 号土地的“蛇形”一键满扫播种，效率极高。
- **智能防卡死**: 独创“动作后清理干扰”逻辑，自动点击界面空白处关闭潜在影响 OCR 的遮挡物。
- **人性化随机模拟**: 所有的点击延时、巡查间隔、坐标选择均加入随机偏移，更贴合人工操作逻辑。
- **自动恢复机制**: 遇到断网、被顶号、活动弹窗时，具备多优先级的自动点击恢复能力。

## 5. 配置参数 (位于 `main.py`)

| 参数名 | 说明 | 示例值 |
| :--- | :--- | :--- |
| `DEFAULT_SEED_NAME` | 默认种植的种子名称 | `"迎春花"` |
| `FRIEND_PATROL_INTERVAL` | 巡查多少次主场后执行一次好友巡查 | `20` |
| `FRIEND_MAX_PATROL` | 单次好友巡查的最大拜访人数 | `5` |
| `LOOP_INTERVAL_MIN/MAX` | 每次大规模巡查之间的随机间隔 | `5-10 秒` |
| `TARGET_SIZE` | OCR 识别前的图像缩放基准尺寸 | `(2000, 3720)` |

## 6. 环境准备

1. **系统环境**: macOS (脚本深度依赖 macOS API)
2. **授权**: 需在“系统设置 -> 隐私与安全性”中授予终端/IDE 以下权限：
   - **屏幕录制** (用于捕捉游戏画面)
   - **辅助功能** (用于模拟鼠标操作)
3. **依赖安装**:
   ```bash
   pip install opencv-python numpy pyautogui pyobjc
   ```

## 7. 使用指南

1. 确保 `QQ经典农场` 窗口已打开且处于可见状态。
2. 在项目根目录下运行：
   ```bash
   python main.py
   ```
3. 观察控制台日志输出。如需停止，按 `Ctrl+C` 即可。

## 8. 目录结构

```
.
├── main.py            # 主逻辑与配置
├── friend.py          # 拜访功能
├── trading.py         # 仓库及商店
├── oc.py              # 图像识别算法
├── README.md          # 本指南
└── templates/         # 存放土地及种子图像模板
    ├── land/          # 土地模板 (黑.png, 红.png, 金.png)
    └── seed/          # 种子模板 (seed_迎春花.png 等)
```

## 9. 注意事项

- **识别精度**: 若识别率下降，请检查窗口是否被缩放，或微调 `oc.py` 中的匹配阈值。
- **模板更新**: 若游戏界面更新（如配色方案改变），请重新截取 `templates/` 下的图标。
- **合规性**: 仅供技术交流学习，请合理排布运行时间，避免高频率操作。
